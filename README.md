# 🌐 Projeto Coffee Website

Versão: 1.0.0

## 📖 Sobre o Projeto

Este projeto consiste no desenvolvimento de um site moderno e responsivo, criado com foco em boas práticas de design, usabilidade e organização de código.

Ele foi criado com fins educacionais e faz parte de um curso de desenvolvimento web.

## 🎯 Objetivo

- Praticar desenvolvimento web moderno.
- Melhorar habilidades em HTML, CSS e responsividade.
- Criar base para um curso completo.
- Evoluir o projeto com novas funcionalidades.

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (opcional)

## 📱 Funcionalidades

- Layout moderno e limpo.
- Design responsivo.
- Navegação intuitiva.
- Estrutura organizada.

## ⚙️ Como Executar

1. Clone o repositório:
git clone https://github.com/tblima26/curso-coffee-website

2. Acesse a pasta:
cd curso-coffee-website

3. Abra o arquivo index.html no navegador

## 📚 Aprendizado

Projeto desenvolvido como parte de um processo contínuo de aprendizado, com objetivo de futuramente se tornar um curso completo de desenvolvimento web.
